# Emmet.Finance UI v2.0

## Pulling

```bash
git pull https://github.com/Emmet-Finance/new-ui.git
yarn
```

## Building

```bash
yarn build
```

## Launching

Development:

```bash
yarn start
```

Production:

```bash
serve -s build
```

If `serve` is not available:

```bash
yarn global add serve
```
